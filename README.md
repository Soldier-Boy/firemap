# FireMap

Karte für Feuerwehren

## Features

Anzeige von:

* Feuerwehrhäuser + Details + Coverage (R=1500m)
* Hydranten + Details + Coverage (R=200m)
* Spitäler + Details

## Demo

[https://rfmtc.no-ip.org/osm/firemap/](https://rfmtc.no-ip.org/osm/firemap/)
